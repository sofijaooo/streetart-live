## Project Overview

StreetArt Live is a client-server application developed to popularize street art and simplify interaction between artists and audiences.

The platform allows:

* Discovering artists and events
* Managing artist profiles
* Submitting event requests
* Moderating events
* Displaying approved events on an interactive map
* Subscribing to artists
* Receiving notifications about upcoming performances

Technology stack:

* Java
* Spring Boot
* PostgreSQL
* REST API
* HTML5
* CSS3
* JavaScript
* Leaflet.js
* JavaFX WebView
* Cloudinary
